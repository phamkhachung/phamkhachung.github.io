Kiểu dữ liệu số và luận lý

Trong lập trình máy tính, một kiểu dữ liệu (data type) là một cách phân loại dữ liệu cho trình biên dịch hoặc thông dịch hiểu các lập trình viên muốn sử dụng loại dữ liệu gì. Trong bài trước chúng ta đã tìm hiểu về các loại chuỗi giá trị và từng loại chuỗi giá trị cũng như kết hợp các chuỗi giá trị như vậy hình thành các kiểu dữ liệu khác nhau trong Python.

Kiểu dữ liệu số

Có ba loại kiểu dữ liệu thuộc nhóm dữ liệu số là:

  • int: số nguyên. Ví dụ: -4, 0, 100, v.v… Độ lớn của số nguyên trong Python không giới hạn theo số bit như các ngôn ngữ khác mà theo bộ nhớ hiện có.
  • float: số thực. Ví dụ: -4.0, 3.14156, v.v… Độ chính xác của số thực đến 15 số sau dấu thập phân.
  • complex: số phức. Ví dụ: -3 + 4j, 3.4 - 5.0j, v.v…

Python cung cấp các toán tử số học phục vụ nhu cầu tính toán cho kiểu dữ liệu số như sau:

Toán tử Ý nghĩa Ví dụ (x = 7, y = 5) Kết quả
+ Phép cộng x + y 12
- Phép trừ x - y 2
* Phép nhân x * y 35
/ Phép chia x / y 1.4
% Phép chia lấy dư x % y 2
** Phép lũy thừa x ** y 16807
// Phép chia lấy phần nguyên x // y 1
& Phép nhân logic bit x & y 5
| Phép cộng logic bit x | y 7
^ Phép cộng loại trừ logic x ^ y 2
~ Phép đảo bit ~x -8
« Phép dịch bit qua trái x « 2 28
» Phép dịch bit qua phải x » 2 1

và các toán tử gán:

Toán tử Ý nghĩa Ví dụ (x = 7, y = 5) Tương ứng Kết quả
= Phép gán bằng x = y x = y x = 5
+= Phép cộng rồi gán lại x += y x = x + y x = 12
-= Phép trừ rồi gán lại x -= y x = x - y x = 2
*= Phép nhân rồi gán lại x *= y x = x * y x = 35
/= Phép chia rồi gán lại x /= y x = x / y x = 1.4
%= Phép chia lấy dư rồi gán lại x %= y x = x % y x = 2
**= Phép lũy thừa rồi gán lại x **= y x = x ** y x = 16807
//= Phép chia lấy phần nguyên rồi gán lại x //= y x = x // y x = 1
&= Phép nhân logic bit rồi gán lại x &= y x = x & y x = 5
|= Phép cộng logic bit rồi gán lại x |= y x = x | y x = 7
^= Phép cộng loại trừ logic bit rồi gán lại x ^= y x = x ^ y x = 2
«= Phép dịch bit trái rồi gán lại x «= 2 x = x « 2 x = 28
»= Phép dịch bit phải rồi gán lại x »= 2 x = x » 2 x = 1

Kiểu dữ liệu luận lý

Kiểu dữ liệu luận lý (bool) cung cấp 2 giá trị là True và False ứng với đúng và sai. Lưu ý những cách viết khác như true, TRUE hay “true” không thể hiện giá trị kiểu dữ liệu này. Thường kết quả các phép toán so sánh là giá trị kiểu bool:

x = 5 Kết quả So sánh Kết quả So sánh Kết quả
x == 5 True x < 5 False x > 5 False
x != 5 False x <= 5 True x >= 5 True

Một số phép toán với bool:

  • not: đảo giá trị, True thành False và False thành True.
  • and: phép logic và: True khi cả 2 vế cùng True, còn lại đều cho giá trị False. Một cách suy kết quả khác là nếu vế đầu là False thì kết quả là False, còn nếu vế đầu là True thì lấy giá trị vế sau làm kết quả cho phép and.
  • or: phép logic hoặc: True khi có ít nhất 1 vế là True. Một cách suy kết quả khác là nếu vế đầu là True thì kết quả là True, còn nếu vế đầu là False thì lấy giá trị vế sau làm kết quả cho phép or.
x = 5 (A) y = 7 (B) not(A) A and B A or B
x < 5 y < 5 True False False
x > 5 y > 5 True False True
x < 7 y < 7 False False True
x <= 5 y >= 7 False True True

Tóm tắt

  • Python cung cấp 3 kiểu dữ liệu số là int cho số nguyên, float cho số thực và complex cho số phức.
  • Python cung cấp đầy đủ các toán tử số học, toán tử bit.
  • Python cung cấp kiểu dữ liệu luận lý là bool có 2 giá trị True và False. Kết quả các phép so sánh sẽ cho giá trị luận lý
  • Python cung cấp các phép toán luận lý not, and và or.
